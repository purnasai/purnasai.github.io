### Best resources for Deep learners, Machine learning , artificial intelligence

There are several or Tons of resources available out there to prepare. but here its going to be minimalistic.

#### Python
All these 4 links here are practice oriented to learn python

- [practice python](https://www.practicepython.org/)

- [w3resource with practice excercise](https://www.w3resource.com/python-exercises/)
